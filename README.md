#DESCRIPTION:  
This shell script is used to sort IP by visit count in the apache access log file.  
**Usage:**  
Run command as:  
```$ ./ip_sort.sh < [LogFile]```  
Then the result will be output to screen.  
***   
#描述:  
这个shell脚本用来对Apache访问日志通过访问量来对IP进行排序。  
**用法:**  
执行如下命令:  
```$ ./ip_sort.sh < [LogFile]```  
之后结果就会输出到屏幕上。
